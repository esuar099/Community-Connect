## macOS Prereqs

- Git
  Should come with macOS already but you can check by running

```
Check install
$ git --version
```

- [Homebrew](https://www.google.com)
- NVM

```
Install
$ mkdir ~/.nvm
$ brew install nvm

Check install
$ nvm --version
```

- Node v16

```
Install
$ nvm install 16.13.0

Check install
$ node -v
$ npm -v
```

- Yarn
```
Install
$ npm i -g yarn

Check install
$ yarn -v
```
